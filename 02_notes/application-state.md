# APPLICATION STATE

An application state is a datatype that defines who can do what and provides the necessary context for interpreting trace data.

- Data type: Structured, relational data with a strict schema
- Structure: Relational and indexed (ie: able to efficiently look up and update data)
- Purpose: Defines who can do what and provides the necessary context for interpreting the trace data
