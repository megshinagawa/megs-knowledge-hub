# ONLINE TRANSACTION PROCESSING

An Online Transaction Processing (OLTP) system allows for consistent access for day-to-day operations. 

- Data structure: Highly **normalized** and relational
- Transaction type: Short, atomic transactions (ACID properties)
- Purpose: Fast, consistent access for day-to-day operations that preserve data integrity
